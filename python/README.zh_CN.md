## 前提条件

- Python 3

## 使用方法

1\. 下载本项目并进入项目目录下的python文件夹

2\. 下载必要的Python包

```bash
pip3 install -r requirements.txt
```

3\. 将无法跳转至上次浏览位置的PDF文件放到当前目录

4\. 运行脚本

```bash
python patch_pdf_for_edge.pdf
```

等待脚本执行结束。
